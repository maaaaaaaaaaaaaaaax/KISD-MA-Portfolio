# KISD MA Portfolio

A modern portfolio website built with React, Vite, TypeScript, and Tailwind CSS.

![Portfolio Preview](https://github.com/user-attachments/assets/f7bd9bd0-042d-4c1d-b741-fe21cdfc2480)

## Tech Stack

- **React 19** - Modern UI library
- **Vite 7** - Lightning-fast build tool
- **TypeScript 5** - Type-safe development
- **Tailwind CSS 4** - Utility-first CSS framework

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

## Development

The development server will start at `http://localhost:5173/` with hot module replacement (HMR) enabled.
